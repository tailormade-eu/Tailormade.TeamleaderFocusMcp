# Migrating

⚠️ WARNING! These endpoints are not guaranteed to stay backwards compatible.

Their only use is to allow older integrations to migrate their data to the new API.

## [📄️ migrate.idTranslates an ID from the deprecated API into a new UUID.](/docs/api/migrate-id)## [📄️ migrate.taxRateTranslates tax rates from the deprecated API into a new UUID tax rate.](/docs/api/migrate-tax-rate)## [📄️ migrate.activityTypeTranslates `meeting`, `call` and `task` into their respective activity type UUID.](/docs/api/migrate-activity-type)
