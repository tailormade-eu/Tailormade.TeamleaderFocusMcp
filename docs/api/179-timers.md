# Timers

Timers are an easy way to track time. You can start, stop or resume a timer any time. However, only one timer can run simultaneously per user. Every action will result in the creation or update of a time tracking.

## [📄️ timers.currentGet the current running timer.](/docs/api/timers-current)## [📄️ timers.startStart a new timer.](/docs/api/timers-start)## [📄️ timers.stopStop the current timer. This will add a new time tracking in the background.](/docs/api/timers-stop)## [📄️ timers.updateUpdate the current timer. Only possible if there is a timer running.](/docs/api/timers-update)
