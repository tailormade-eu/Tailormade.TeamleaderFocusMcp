# Legacy Milestones

Every projects consists of one or more milestones which are limited in time and budget. Budget is however not included in this endpoints (yet).

_Required scopes:`projects`_

## [📄️ milestones.listGet a list of project milestones.](/docs/api/legacy-milestones-list)## [📄️ milestones.infoGet details for a single milestone.](/docs/api/legacy-milestones-info)## [📄️ milestones.createCreate a new milestone.](/docs/api/legacy-milestones-create)## [📄️ milestones.updateUpdate a milestone.](/docs/api/legacy-milestones-update)## [📄️ milestones.deleteDelete a milestone.](/docs/api/legacy-milestones-delete)## [📄️ milestones.closeClose a milestone. All open tasks will be closed, open meetings will remain open. Closing the last open milestone will also close the project.](/docs/api/legacy-milestones-close)## [📄️ milestones.open(Re)open a milestone. If the milestone's project is closed, the project will be reopened.](/docs/api/legacy-milestones-open)
