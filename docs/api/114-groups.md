# Groups

_**Required scopes:**_ `projects`

[

## 📄️ projectGroups.list

Lists all the groups that match the optional filters provided.

](/docs/api/project-groups-list)

[

## 📄️ projectGroups.info

Returns all the information for one group.

](/docs/api/project-groups-info)

[

## 📄️ projectGroups.create

Create a group. All properties except for \`title\` and \`project\_id\` are optional.

](/docs/api/project-groups-create)

[

## 📄️ projectGroups.update

Update a group. All attributes except for \`id\` are optional. Providing \`null\` will clear that value from the project (for properties that are nullable).

](/docs/api/project-groups-update)

[

## 📄️ projectGroups.duplicate

Duplicate a group and its entities, without any time trackings.

](/docs/api/project-groups-duplicate)

[

## 📄️ projectGroups.delete

Delete a group.

](/docs/api/project-groups-delete)

[

## 📄️ projectGroups.assign

Assign a user or a team to a group.

](/docs/api/project-groups-assign)

[

## 📄️ projectGroups.unassign

Unassign a user or a team from a group.

](/docs/api/project-groups-unassign)
