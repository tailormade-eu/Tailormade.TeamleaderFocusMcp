# Calendar events

Calendar events are scheduled events in your calendar. A calendar event involves a particular activity type: a task, meeting or call.

_**Required scopes:**_ `events`

## [📄️ events.listGet a list of calendar events.](/docs/api/events-list)## [📄️ events.infoGet details for a single calendar event.](/docs/api/events-info)## [📄️ events.createCreate a new calendar event.](/docs/api/events-create)## [📄️ events.updateUpdate a calendar event.](/docs/api/events-update)## [📄️ events.cancelCancel a calendar event (for all attendees).](/docs/api/events-cancel)
