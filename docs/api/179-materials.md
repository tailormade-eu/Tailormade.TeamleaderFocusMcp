# Materials

_**Required scopes:**_ `projects`

[

## 📄️ materials.list

Lists all the materials that match the optional filters provided.

](/docs/api/nextgen-projects-materials-list)

[

## 📄️ materials.info

Returns all the information for one material.

](/docs/api/nextgen-projects-materials-info)

[

## 📄️ materials.create

Create a material. All properties except for \`title\` and \`project\_id\` are optional.

](/docs/api/nextgen-projects-materials-create)

[

## 📄️ materials.update

Update a material. All attributes except for \`id\` are optional. Providing \`null\` will clear that value from the project (for properties that are nullable).

](/docs/api/nextgen-projects-materials-update)

[

## 📄️ materials.duplicate

Duplicate a material.

](/docs/api/nextgen-projects-materials-duplicate)

[

## 📄️ materials.delete

Delete a material.

](/docs/api/nextgen-projects-materials-delete)

[

## 📄️ materials.assign

Assign a user or a team to a material.

](/docs/api/nextgen-projects-materials-assign)

[

## 📄️ materials.unassign

Unassign a user or a team from a material.

](/docs/api/nextgen-projects-materials-unassign)
