# API download method — Teamleader Focus

**Bron:** https://developer.focus.teamleader.eu/
**Type:** HTML SPA (client-side rendering, geen formele OpenAPI publiek)
**Tool:** Custom scraper (nog te bouwen — bestaande `docs/api/*.md` zijn ad-hoc gegenereerd in eerdere sessie)

## Run

```bash
npm run scrape-api              # full scrape (overschrijf alles)
npm run scrape-api:discover     # alleen endpoints tonen (geen schrijf)
npm run scrape-api:diff         # alleen ontbrekende endpoints scrapen
```

Script: `scripts/scrape-tl-api.ts` — Playwright headless Chromium, transitive BFS crawl (CONCURRENCY=5), Turndown HTML→MD.
