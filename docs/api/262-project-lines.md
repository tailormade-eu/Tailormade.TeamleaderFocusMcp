# Project lines

> A project line can be a group, a task, or a material.

> The `projectLines.list` endpoint only returns `{id, type}` pairs of those lines.

_**Required scopes:**_ `projects`

## [📄️ projectLines.listAll line types (tasks, materials, groups) can be side-loaded.](/docs/api/project-lines-list)## [📄️ projectLines.addToGroupAdd an existing task or material to a group.](/docs/api/project-lines-add-to-group)## [📄️ projectLines.removeFromGroupRemove a task or material from the group it is currently in.](/docs/api/project-lines-remove-from-group)
