# Deal Phases

Deal phases are the different phases a deal typically goes through. A deal can be moved to either a next or previous phase. A deal can be marked as won or lost independently from the deal phase.

_**Required scopes:**_ `deals`

## [📄️ dealPhases.listGet a list of all phases a deal can go through, sorted by their order in the flow.](/docs/api/deal-phases-list)## [📄️ dealPhases.createCreate a new deal phase.](/docs/api/deal-phases-create)## [📄️ dealPhases.updateUpdate a deal phase.](/docs/api/deal-phases-update)## [📄️ dealPhases.duplicateCreate a new deal phase by duplicating an existing one.](/docs/api/deal-phases-duplicate)## [📄️ dealPhases.moveMove a phase to a new position in the pipeline.](/docs/api/deal-phases-move)## [📄️ dealPhases.deleteDelete a phase.](/docs/api/deal-phases-delete)
