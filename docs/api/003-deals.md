# Deals

Deals are sale opportunities, which need to be followed up by their responsible user. A deal can move through specific deal phases, and be marked as won or lost.

_**Required scopes:**_ `deals`

## [📄️ deals.listGet a list of deals.](/docs/api/deals-list)## [📄️ deals.infoGet details for a single deal.](/docs/api/deals-info)## [📄️ deals.createCreate a new deal for a customer.](/docs/api/deals-create)## [📄️ deals.updateUpdate a deal.](/docs/api/deals-update)## [📄️ deals.moveMove the deal to a different phase.](/docs/api/deals-move)## [📄️ deals.winMark a deal as won.](/docs/api/deals-win)## [📄️ deals.loseMark a deal as lost.](/docs/api/deals-lose)## [📄️ deals.deleteDelete a deal.](/docs/api/deals-delete)## [📄️ lostReasons.listGet a list of lost reasons for deals.](/docs/api/lost-reasons-list)
